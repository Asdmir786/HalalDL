# v0.4.1 - The Precision Polish Update

Released: 2026-04-12

## ⚡ The "Precision Polish" Update!

HalalDL v0.4.1 is a focused quality release for the workflows people are already using: faster quick downloads, settings that actually stick, clearer finished results, and more control over filenames and clip ranges. This one is small on version number, but it smooths out the exact places where daily use can feel annoying.

## ✨ What’s New in v0.4.1?

### 🎯 Better finished download cards

* **Total output size**: Finished downloads now show the final output size directly on the card. If the job produced subtitles or multiple sidecar files, HalalDL totals the outputs instead of only showing one file.
* **Media duration**: Video and audio jobs can now show their timeline length, so a download card can tell you whether something is a short clip, a full video, or a longer session.
* **Image-aware details**: Image-only downloads stay clean. HalalDL does not show a timeline for media that has no timeline.

![Download details - light](https://raw.githubusercontent.com/Asdmir786/HalalDL/main/docs/assets/releases/0.4.1/promo/download-details-light.png)
![Download details - dark](https://raw.githubusercontent.com/Asdmir786/HalalDL/main/docs/assets/releases/0.4.1/promo/download-details-dark.png)

### ⚡ Cleaner quick downloads

* **Compact quick panel (#6)**: The quick panel has been tightened so repeated save location, start mode, subtitles, and description details do not eat the whole panel.
* **Keyboard-friendly flow**: The panel is easier to use without reaching for the mouse, especially on smaller screens.
* **Less clutter, same control**: Important choices are still present, but the main download action stays easier to reach.

![Compact quick panel - light](https://raw.githubusercontent.com/Asdmir786/HalalDL/main/docs/assets/releases/0.4.1/promo/quick-panel-light.png)
![Compact quick panel - dark](https://raw.githubusercontent.com/Asdmir786/HalalDL/main/docs/assets/releases/0.4.1/promo/quick-panel-dark.png)

### 🧩 Preset filename templates

* **Preset-level naming (#5)**: Custom presets can now carry their own filename template, so a preset can define how files should be saved without requiring per-download overrides.
* **Safer extension handling**: If a template leaves out `%(ext)s`, HalalDL adds it at download time to avoid extensionless files.
* **Works across download paths**: The template handling applies to normal yt-dlp downloads and the Instagram fallback path.

![Preset filename templates - light](https://raw.githubusercontent.com/Asdmir786/HalalDL/main/docs/assets/releases/0.4.1/promo/preset-filenames-light.png)
![Preset filename templates - dark](https://raw.githubusercontent.com/Asdmir786/HalalDL/main/docs/assets/releases/0.4.1/promo/preset-filenames-dark.png)

### ✂️ Clip download mode

* **Start and end fields**: Timed media downloads can now use optional clip start/end values.
* **Flexible time input**: Use seconds, `mm:ss`, or `hh:mm:ss`.
* **Guardrails included**: HalalDL blocks invalid clip ranges and avoids showing clip behavior for image-only downloads.

### ✨ Better latest-result spotlight

* **Finite glow animation**: Notification-routed results now get a proper animated glow that finishes instead of glowing forever.
* **Latest marker**: The newest finished result gets a subtle marker so it is easier to tell which card just completed.
* **More accurate attention state**: Older opened results no longer pretend to be the newest result.

![Latest result spotlight - light](https://raw.githubusercontent.com/Asdmir786/HalalDL/main/docs/assets/releases/0.4.1/promo/latest-glow-light.png)

## 🐛 Fixes & Improvements

* **Settings persistence (#7)**: Fixed settings sometimes reverting after leaving Settings, including tray left-click and tray double-click behavior.
* **Settings saving**: Settings changes now commit automatically instead of depending on a draft state that could be lost during navigation.
* **React hooks cleanup**: Fixed lint issues in the new attention/spotlight paths.
* **Storage cleanup**: Removed a leftover startup console log.
* **Release safety**: Completed lightweight release checks without running build-style tests.

## 📦 Choosing Your Version

* **Full (Recommended)**: Best for most users. Includes all dependencies such as FFmpeg and yt-dlp pre-configured.
* **Lite**: Best for power users who manage their own tools.

## ⚠️ Known Notes

* **Clip mode depends on yt-dlp behavior**: Clip downloads use yt-dlp download sections, so exact cut behavior can depend on the source, format, and available tooling.
* **Duration depends on metadata**: Duration appears when HalalDL can get reliable media metadata. Some sources may not expose it.
* **Release images include separate themes**: The v0.4.1 asset pack includes dedicated light and dark PNGs instead of mixed-theme screenshots.

## ⬆️ Upgrade Notes

* Existing settings, presets, downloads, and history should carry forward normally.
* If you previously changed tray behavior and it did not stick, set it once after updating to v0.4.1 and it should persist normally.
* Custom preset filename templates do not require `%(ext)s`; HalalDL will add the extension token when needed.

## 📝 Note from the Developer

This release exists because people are actually using HalalDL and reporting the little things that matter. The goal for v0.4.1 is simple: make the daily flow tighter, make finished downloads more informative, and fix the settings behavior before it annoys anyone else.
