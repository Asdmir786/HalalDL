# Release Checklist (v1)

## Before Release
- All screens functional
- Tool detection tested
- Lite vs Full behavior verified
- Logs export verified
- Preset import/export tested

## GitHub Release
- Two assets:
  - HalalDL-Lite-Windows
  - HalalDL-Full-Windows
- Release notes include:
  - Known limitations
  - Required tools
  - Future roadmap

## Post-Release
- Collect feedback
- Log common failures
- Prepare v1.1 improvements
