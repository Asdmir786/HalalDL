# UI Specification (v1)

## Main Screens
1. Downloads
2. Presets
3. Tools Manager
4. Logs & Diagnostics
5. Settings

---

## Downloads Screen

### Top Section
- URL input
- Add to Queue button
- Preset selector:
  - Default: Global preset
  - Allows per-download override

### Queue
- Virtualized list
- Each job shows:
  - Title (when resolved)
  - Progress bar
  - Speed / ETA
  - Status (Queued/Downloading/Post-processing/Done/Failed)
  - Actions: Pause, Resume, Cancel, Open Folder

### Queue Behavior
- Default concurrency: 1
- User configurable up to 5
- Drag-and-drop reorder supported

---

## Presets Screen

### Built-in Presets
- Read-only
- Can be duplicated (creates an editable copy)

### User Presets
- Editable
- Can be deleted
- Import/export JSON supported

### Preset Application Rules
- One global default preset
- Per-download override allowed
- When a job is queued, it stores a snapshot of the selected preset settings

---

## Logs & Diagnostics

### Live Logger
- Shows full raw yt-dlp output
- Stdout + stderr merged
- Raw is the default view (no hiding logs)

### Controls
- Copy yt-dlp command button
- Export logs button

### Export Contents (zip)
- app-info.json (app version, OS, key settings summary)
- tools-status.json (tool modes, paths, versions detected)
- job.json (job settings snapshot + selected preset)
- yt-dlp.log (full raw output)

---

## Settings

### General
- Theme (dark/light)
- Notifications on/off
- Notify on complete and/or fail

### Download Behavior
- Concurrency
- Retry count (default 2; user configurable)

### File Collision
- Skip existing (default)
- Auto-rename
- Overwrite (warn)
