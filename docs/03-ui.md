# UI Specification (v1)

## Main Screens
1. Downloads
2. Presets
3. Tools Manager
4. Logs & Diagnostics
5. Settings

---

## Downloads Screen

### Top Section
- URL input
- Add to Queue button
- Preset selector:
  - Default: Global preset
  - Allows per-download override

### Queue
- Virtualized list
- Each job shows:
  - Title (when resolved)
  - Progress bar
  - Speed / ETA
  - Status
  - Actions: Pause, Resume, Cancel, Open Folder

### Queue Behavior
- Default concurrency: 1
- User configurable up to 5
- Drag-and-drop reorder

---

## Presets Screen

### Built-in Presets
- Read-only
- Can be duplicated

### User Presets
- Editable
- Can be deleted
- Import/export JSON supported

### Preset Application
- One global default preset
- Per-download override allowed
- Job stores a snapshot of preset settings at queue time

---

## Logs & Diagnostics

### Live Logger
- Shows full raw yt-dlp output
- Stdout + stderr merged
- No filtering by default

### Controls
- Copy yt-dlp command button
- Export logs button

### Export Contents
- App info
- Tool versions
- Job settings snapshot
- Full raw log

---

## Settings

### General
- Theme (dark/light)
- Notifications on/off
- Notify on complete/fail

### Download Behavior
- Concurrency
- Retry count (default 2)

### File Collision
- Skip existing (default)
- Auto-rename
- Overwrite (warn)
